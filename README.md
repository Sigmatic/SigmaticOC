## Installation

SigmaticOC is available through [CocoaPods](http://cocoapods.org). To install the library, simply add the following line to your Podfile:
```ruby
pod "SigmaticOC"
```
<br />
Or if you are only interested in a subspec, use 
<br />
```ruby
pod "SigmaticOC/<Subspec>"
//example
pod "SigmaticOC/JSON"
```
##### Available Subspecs
- Array
- Bool
- Date
- Dictionary
- JSON
- Number
- Protocol
- String
